# TOEFL Typing Practice App

一个面向 TOEFL 备考场景的英语输入训练应用，聚焦两类核心能力：

1. 作文 section 的打字准确性。
2. 词汇拼写与回忆能力。

项目的目标不是简单跟打，而是通过动态生成练习内容、错题回流和计时挑战，让每次训练都尽量不同，并逐步贴近用户自己的薄弱点。

## 核心模式

### 1. 作文打字模式

用于训练长文本输入能力，重点关注：

- 单词拼写
- 标点
- 大小写
- 漏字、多字、错字
- 长句输入稳定性

### 2. 词汇拼写模式

用于训练单词拼写和记忆回忆，支持：

- 中文释义 -> 英文单词
- 英文释义 -> 单词
- 例句挖空
- 词根词缀提示

### 3. 计时挑战模式

作文打字模式下的子模式，用于在限定时间内完成更多内容或条目，同时兼顾准确率。

## 内容来源

练习素材会优先围绕以下方向生成和组织：

- 邮件场景
- 学术讨论
- TOEFL 必备词汇库
- TOEFL 常见写作表达
- 常见标点与长难句结构

## 设计原则

- 每次练习尽量不同，而不是固定文本重复。
- 使用受约束的随机生成，而不是纯随机。
- 把用户高频错误回流到后续练习中。
- 通过难度分层，让训练节奏可控。
